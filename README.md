## Project Overview
This project analyzes StayCatin Airbnb business performance using Tableau. The dashboard provides insights into revenue trends, booking patterns, and property performance.

## Objectives
• Analyze revenue and booking trends  
• Identify top-performing properties  
• Track occupancy and availability  
• Provide business insights using interactive dashboard  

## Key Insights
• Identified highest revenue-generating properties  
• Analyzed booking trends over time  
• Evaluated occupancy rate performance  
• Provided business-level KPI overview
